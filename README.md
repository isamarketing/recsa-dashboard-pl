# RECSA — Dashboard P&L por Producto

Dashboard interactivo de Pérdidas & Ganancias por producto para RECSA.

## Acceso
https://isamarketing.github.io/recsa-dashboard-pl/

## Características
- 6 vistas: Resumen Ejecutivo, Por Línea, Por Producto, Alertas, Simulador de Pricing, Vista Unitaria
- Parámetros editables en vivo (TC, volumen, gastos fijos, MOD, impuestos, TNA)
- Carga dinámica desde Excel Master — productos nuevos aparecen automáticamente
- Export CSV

## Actualización
1. Actualizar el Excel Master consolidado (RECSA_MASTER_PG_INPUT.xlsx)
2. Abrir el dashboard y hacer click en "Cargar Excel Master"
3. Arrastrar el Excel actualizado
